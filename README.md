# 🌊 DWCI Pro – Depth Weighted Water Capture Index

Herramienta desarrollada en ArcGIS Pro para generar un índice de captación hídrica basado en análisis raster y ponderación multicriterio.

---

## 📌 Descripción

DWCI Pro permite identificar zonas con mayor potencial de captación de agua mediante la integración de variables ambientales clave.

---

## 🧰 Requisitos

- ArcGIS Pro  
- Extensión Spatial Analyst  

---

## 📊 Variables utilizadas

- TWI (Topographic Wetness Index)  
- Geología  
- Uso de suelo  
- Pendiente  
- Profundidad de pozos  

---

## ⚙️ Flujo de trabajo

1. Cargar el toolbox en ArcGIS Pro  
2. Ingresar los raster  
3. Definir los pesos  
4. Ejecutar la herramienta  
5. Analizar el resultado
